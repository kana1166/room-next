会議室予約システム

概要

このプロジェクトは、会議室の予約管理を行うための Web アプリケーションです。ユーザーは複数の会議室を予約し、管理者は全ての予約を監視し管理することができます。予約システムでは会議室の写真閲覧、予約キャンセル規則、役員専用会議室の予約などの機能を提供します。

技術スタック

Frontend: Next.js
Backend: FastAPI
Data Visualization: Streamlit
Database: MySQL/PostgreSQL
